🌧️ MEGHDOOT - Smart Irrigation & Monitoring System

An IoT-based smart irrigation system designed to improve water management and energy efficiency in agriculture.
MEGHDOOT uses sensors, microcontrollers, cloud connectivity, and AI to automate irrigation and monitor farm conditions in real time.

🚀 Features
Core System

Real-time farm monitoring using IoT sensors

Automated irrigation control

AI-based irrigation decision system

Rainwater monitoring integration

Energy optimization for irrigation pumps

Sensor Monitoring

Soil moisture monitoring

Humidity monitoring

Rainfall detection

Water tank level monitoring

Dashboard

Real-time sensor data visualization

Irrigation status monitoring

Alerts and notifications

Historical data tracking
